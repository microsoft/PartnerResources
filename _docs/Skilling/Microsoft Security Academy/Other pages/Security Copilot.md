#### Considerations

* **Assist a Human in Completing Work** â€“ It's a Copilot, integrations are driven by/drive human engagement, not background runtime processing of substantial amounts of data.
* **Have High Customer Value** â€” The cost of Generative AI is orders of magnitude per transaction higher than your average feature today and depend on a constrained hardware supply (GPUs).
* **Will be Regularly Used** â€” The best integrations will be used regularly so it is ongoing value, not a one-time value (like a configuration assistant).
* **Work with Constrained Context**Â â€” The model has a token limit for both input and output for the entire session.
* **Leverage Model Creativity** â€” The model is probabilistic and not deterministic; the best integrations are cases where that is additive to the user experience.
